C’est quoi ?

Atom est un éditeur de texte multi-plateforme développé par Gihub. Son plus grand atout est qu’il est hautement configurable et personnalisable grâce à un puissant système de paquets (extensions). Atom n’est donc pas livré seul, une multitude de paquets s’y ajoute pour rendre son utilisation plus confortable et productive.

Comment on l’utilise ?

Un raccourci Atom est normalement disponible sur le bureau. Son démarrage est un peu long. Pour ouvrir un fichier ou un dossier ça se passe dans le menu : File → Open File ou Open Folder ou encore Add Project Folder (mon préféré). Pour configurer Atom, que ce soit son comportement, son thème ou un paquet : Edit → Preferences. Pour ouvrir le manuel en ligne : Help → Documentation. Toutes les actions liées aux paquets sont dans le menu : Packages.

Les paquets

Pour rappel les paquets sont comme des extensions qui peuvent ajouter des fonctionnalités ou simplement modifier l’apparence.

BDD

data-atom

Permet d’exécuter des requêtes SQL directement dans Atom (au revoir phppgadmin).
D’abord il faut se connecter au serveur en ouvrant un panneau. Il est lié au fichier courant, il faut donc se reconnecter pour chaque nouveau fichier.
Pour ouvrir le panneau : Packages → Data Atom → Toggle results view ou Alt+Shift+R
Ensuite on clique sur le bouton +. Une boite de dialogue apparaît, dans connexion on choisit IUT ; les informations se complètent automatiquement sauf votre mot de passe qu’il faudra rentrer. Pour finir on clique sur Connect.
Pour lancer une requête il suffit de la sélectionner/surligner et d’appuyer sur le bouton Execute ou F5
