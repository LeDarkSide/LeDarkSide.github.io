Présentation

The Dark Side est un mini projet qui a pour but de faciliter l’utilisation de l’environnement Linux installé sur les PC de l’IUT d’Arles.
Son principe est simple : une fois connecté à votre session il suffit de lancer le script TheDarkSide.bash. Il va d’abord télécharger un fichier compressé qui contient, entre autre, plusieurs scripts. Après l'avoir extrait, il installe les logiciels pour travailler. Enfin il reconfigure l’environnement de travail.
Lorsqu’une mise à jour sera disponible, une boite de dialogue vous demandera si vous voulez l’installer.
Il est possible de modifier le comportement de The Dark Side pour qu’il puisse correspondre au mieux à vos attentes.

Installation

The Dark Side doit être installé sur chaque nouveau PC via le script TheDarkSide.bash. C’est pourquoi il est recommandé de mettre le script dans le net-home afin d’y accéder plus facilement.

En une seule commande

Dans un terminal lancez :

curl -sL -o ~/net-home/TheDarkSide.bash "https://raw.githubusercontent.com/L0L022/TheDarkSide/master/TheDarkSide.bash" && chmod u+x ~/net-home/TheDarkSide.bash && bash ~/net-home/TheDarkSide.bash

À la main

Le lien vers le script à télécharger : TheDarkSide.bash
Il faut le rendre exécutable en faisant clic droit → Propriétés → Permissions → Autoriser ce fichier à être exécuté comme un programme
